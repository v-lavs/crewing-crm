BEST SOLUTION FOR CREW AGENCIES

CrewingCRM is the solution of two main problems for the maritime crewing agencies such as gathering and organization of information (sailors, ship owners and other members), optimization and automation of the working process in general.

  • Custom Printed Forms

    We offer a large selection of custom printed forms that are printed with your company information. Our custom forms are affordable and include contracts, crew lists, various applications and more!

  • Document Storage and Monitoring

    You can store any kind of certificates or documents without any restrictions. In addition you will get the ability to track expiry status for all certificates in the CrewingCRM database. Also, you will be able to customize alerts according to your requirements.

  • Role-Based View

    Security is a cornerstone of CrewingCRM and with the passing time, we evolving new security features of CrewingCRM. This feature provides the following abilities: hide/ Show views for a user for a particular entity based on their security role, set the default view for a user for a particular entity based on security role.

  • Public Application Form

    Our online application form allows you receiving new applications directly into the CrewingCRM database, eliminating the hassle of collecting MS Word, MS Excel or pdf documents.

  • Advanced Search

    Advanced search will help you to find the most appropriate candidate for replacing existing crew onboard or filling in new vacancies. Finding right seaman will be easy and fast thanks to the detailed criteria filtering options. You can run the search by personal information, sea services, documents and certificates.
